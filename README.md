# Mauli Tractor App

ट्रॅक्टर सेवा व्यवस्थापनासाठी React + Firebase Web Application.

## Features:
- चालक लॉगिन (नाव आणि मोबाईल)
- 10 प्रकारची यंत्रे (Tools)
- शेतकरी नोंदणी आणि बिलिंग
- ॲडमिन पॅनेल (सर्व रेकॉर्ड पाहणे, संपादन, हटवणे)
- बिलिंग सिस्टम (प्रिंट, WhatsApp शेअर)

## Setup:

1. Clone repository
2. `npm install`
3. `npm run dev` (development)
4. `npm run build` (production)

## Firebase Configuration:
App.jsx मध्ये Firebase config टाका.

## Deploy on Vercel:
1. GitHub वर push करा
2. Vercel मध्ये नवीन प्रोजेक्ट तयार करा
3. GitHub repository निवडा
4. Deploy!
